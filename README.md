# ai4eo_final_project
## Overview
This project focus on unsupervised anomaly detecting in agricultural fields using Sentinel-2 multispectural imagery and Torchgeo.

The goal of this project is to:
1. detect anomalous crop patterns
2. provide interpretable zoning maps

- src/ dataloaders, models
- results/ anomaly_map.png, confusion_matrix.png
